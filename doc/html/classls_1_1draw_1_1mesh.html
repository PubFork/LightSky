Detailed Description

Mesh Object.

A mesh object contains information about the layout of vertex data located on the GPU. It's designed to be used directly with OpenGL. These classes are rather large. It is recommended to instantiate them using

dynamic allocation.

const std::string &  str

Initialize.

Similar to its overload, this method sends text/string data to the GPU using a texture atlas and a corresponding string object. This method will generate a mesh on the fly, attempting to match the characters in the input string, and send the vertices to the GPU.

Parameters
glyphAtlasA texture Atlas containing pre-loaded font glyphs and their offsets.
strA constant reference to a character string, containing data which is to be loaded onto the GPU.
